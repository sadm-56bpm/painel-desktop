ⓘSobre o Consultor SADM

Consultor SADM é o assistente conversacional institucional da SADM. Esta é a versão local v1.0: busca semântica sobre a base de conhecimento canônica embarcada. Próxima versão (32.B) conectará Multi-IA Gratuita (DA-067) via proxy Apps Script com RAG anonimizado (DA-070).

⚠ Esta versão responde apenas sobre arquitetura institucional, processos, governança e normas. Não acessa dados pessoais nem documentos sigilosos (DA-003 ACESSO_RESERVADO).
